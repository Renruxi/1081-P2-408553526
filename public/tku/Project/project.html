    This Is My Project

    本次期末Project是繼承了期中Project，etrain為主題，利用後半學期所學知識： 將後端MySQL2數據庫的數據抓到前端網頁中，在網頁中展示出來。 這個過程還蠻艱辛的。因為在demoController.js中犯了粘貼剪切的錯誤，沒有把demo定義， 而是定義了數據庫。因此在接下來的連接數據的過程中一直沒有連到。 經過求助老師，得知自己在ejs中的路徑寫錯了。也明白了ejs檔裡面是不可以添加靜態 網頁的路徑的，是需要用到相對路徑。
